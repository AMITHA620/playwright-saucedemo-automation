Playwright Automation Test Cases

1. Valid Login Test
Verify user can login with correct username and password.

2. Invalid Login Test
Verify error message appears for incorrect credentials.

3. Product Page Validation
Verify product list is displayed.
Verify 6 products are visible.
Verify each product has image, title, price, and Add to Cart button.

4. Add to Cart Test
Login to application.
Add two products to cart.
Verify cart count is updated.
Verify items appear in cart page.

5. Checkout Test
Add product to cart.
Enter checkout details.
Complete purchase.
Verify confirmation message is displayed.
